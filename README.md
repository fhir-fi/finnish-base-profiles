# Finnish FHIR Base Profiles

This repo contains the source code for the Finnish FHIR base profiles.

The profiling work is performed in a project driven by [HL7 Finland](https://www.hl7.fi). See the [announcement](https://www.hl7.fi/hl7-fhir-profilointityo-kaynnistyy-tule-mukaan-vaikuttamaan-kansalliseen-tekemiseen/), the [running memo](https://docs.google.com/document/d/1yNq6XMLhWJqi6OELQtWC1DFwdtD9CQulzVOfz-zZCko/edit#) and [some more details](https://fhir.fi).
